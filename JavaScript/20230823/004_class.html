자바스크립트 만의 사용자 정의 타입 생성(객체 생성) 방법을 다른 언어의 클래스 무넙처럼 바꿔준 것이 바로 자바스크립트의 클래스(classes)입니다 위의 코드에서 객체지향의 중요한 개념중 하나인 인캡슐레이션을 만나볼 수 있습니다. 인캡슐레이션은 객체지향 프로그래밍의 개념 중 하나로, 데이터와 해당 데이터를 조작하는 메서드를 하나의 단위로 묶는 것을 말합니다. 이처럼 내부적인 동작은 동일하지만 더 보기좋고 편리하게 개선된 문법을 Syntactic sugar라고 부릅니다. (달달하다~) classes 문법은 단순한 설탕 문법은 아닙니다. 여러가지 추가된 문법이 있습니다. (단순히 생성자함수를 바꿨다고만 볼 수는 없음) 실습 - 생성자 함수를 class 문법으로 바꾸어 봅시다!
