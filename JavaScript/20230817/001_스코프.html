JS의 스코프

전역 스코프

  • 스크립트의 어디서든 접근이 가능하기 때문에 사용이 쉽습니다.
  • 타인과의 협업 라이ㅡ러리 사용시 충돌의 위험이 있습니다.

    • 함수 스코프

  • 함수 외부에서 접근이 불가능함
    •         function hello(a){
            const val = "goood"
            return a + "hello" + val
        }
        hello("sohee");
        val // > error

    블록 스코프

  • if 문과 for 문의 {} 블록스코프 안에서 선언된 변수들은 이 안에서만 생존이 가능합니다. 외부에서 이 변수들로 접근이 불가능함 = 외부에서 알아내고 싶어서 찍어봐도 알 수가 없다.
  • 블록스코프는 ES6이후에 추가된 문법이기 때문에 이 이전에 만들어진 변수형인 var키워드는 블록스코프가 적용되지 않으므로 if문과 for문 밖에서 접근 가능함 
    • 
        // 블록 레벨 스코프
        if(true){ // for문이어도 마찬가지입니다.
            let x = 10;
            const y = 10;
        }
        console.log(x, y)

        // 밖에 선언된 x는 함수 내부에서도 접근 가능합니다.
        let x = 100
        function xplus(){
            x = x + 10
        }

        xplus()
        console.log(x)

        // 함수가 종료된 다음에는 선언된 변수는 휘발됩니다.
        function xplus(){
            let a = 100
            a = a + 10
        }

        xplus()
        console.log(a) // Error

        // 함수 안에 함수
        function a(){
            console.log('a 실행')
            function b(){
                console.log('b 실행')
            }
            b()
        }

        a()
        b() // Error
