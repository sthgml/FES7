생성자함수 Constructor

- 생성자 함수를 사용하는 이유 - 동일한 프로퍼티를 가지는 객체 생성 - prototype을 이용하여 메모리 효율을 높일 수 있음 
        const Cook = function (one, two, three, four) {
            this.one = one
            this.two = two
            this.three = three
            this.four = four
        }

        const cook1 = new Cook(1, 2, 3, 4)
