재귀함수

        function factorial (n) {
            if (n <= 1) {
                return n
            }
            return n * factorial(n - 1)
        }

        console.log(factorial(5))
실행 순서 
        factorial(5) 실행
        5 = 1보다 작거나 같지 않으므로 if 뛰어넘고
        5 * factorial(4)를 리턴하려고
        factorial(4) 실행
        4 = 1보다 작거나 같지 않으므로 if 뛰어넘고
        4 * factorial(3)를 리턴하려고
        factorial(3) 실행
        3 = 1보다 작거나 같지 않으므로 if 뛰어넘고
        3 * factorial(2)를 리턴하려고
        factorial(2) 실행
        2 = 1보다 작거나 같지 않으므로 if 뛰어넘고
        2 * factorial(1)를 리턴하려고
        factorial(1) 실행
        1 = 1보다 같거나 작으므로 1을 리턴함

        1을 리턴받은 factorial(1) 
        2* (factorial(1)=1) = 2
        2를 리턴받은 factorial(2)
        3* (factorial(2)=2) = 6
        6을 리턴받은 factorial(3)
        4* (factorial(3)=6) = 24
        24를 리턴받은 factorial(4)
        5 * (factorial(4)=24) = 120
        120을 리턴받은 factorial(5)
