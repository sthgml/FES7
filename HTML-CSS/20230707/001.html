CSS 언어에는 충돌 시 어떤 규칙이 우선하는지 규칙이 있습니다
  1. 후자 우선의 원칙
    1. CSS 상 더 밑에 선언된 것이 적용됩니다.
  2. 구체성(명시도)의 원칙
    1. 가중치 : 개발자 도구의 '계산됨' 탭에서 확인해보기
      1. 1000점 inline style property
      2. 100점id #
      3. 10점 class ., 가상클래스 속성선택자
      4. 1점type(tag), 가상요소 선택자
        5. 명시도 계산기 사이트
  3. 중요성의 원칙
    1. !important
      1. !important 선언 위치 : 선택자 { 속성 : 속성값 (여기) ;}
      2. 이 선언은 다른 CSS의 어떤 선언 보다도 우선합니다.
      3. !important 는 선택자 우선순위에 직접적인 영향을 미칩니다.
      4. important 사용은 좋지 못한 습관입니다. CSS의 자연스러운 상속을 깨트리기 때문에 오류/버그 발생 시 수정을 어렵게 만듭니다.
      5. !important 도 여러개 사용하면 후자가 우선순위로 적용
